container

This section contains 2 functions for building and displaying SVG containers. Each map built with the geoviz library will start with the init function and end with the render function.

container
Static Members
▸ create(arg)
src/container/create.js

The create function is the first step in map construction. t creates an svg container into which the various layers can be added.

create(arg: object): SVGSVGElement
Parameters
arg (object = {}) options and parameters
Name Description
arg.height number (default null) height of the container
arg.projection (function | string) (default "none") d3 function of projection. See d3-geo & d3-geo-projection . You can aslo write "mercator" tu use tiles. (default: "none")
arg.domain (object | Array<object>) the domain corresponds to the geographical area to be displayed. It is defined by a geoJSON or an array containing geoJSONs. By default, the entire world is represented.
arg.pos Array<number> (default [0,0]) position of the container (if contained in another svg container)
arg.background string (default "none") background color
arg.width number (default 1000) width of the container. This value is automatically calculated according to domain . But it can be forced by entering a value.
arg.margin (number | Array<number>) (default [0,0,0,0]) margins around the map. A number to set the same margin everywhere or an array [top, right, bottom, left] to set different margins.
arg.parent object (default null) name of parent container into which this child container is to be included. In this case, the options.pos parameter is also used.
arg.fontFamily string (default "Arial") font-family for the entire map
arg.zoomable (boolean | number | string) (default false) activates the map zoom function. If you set an array of 2 values, it defines the scaleExtent (default: [1,8]). Use "versor" to activate versor zoom . "versor" is only available for vector geometries in wgs84.
arg.control (boolean | Array<number>) (default true) If zoomable is enabled, set the control parameter as true displays control buttons to zoom on the map. You can also define an array of 2 values to locate the panel in the position you want (e.g. [100, 200]). This setting is not available with the Versor zoom.
arg.warning boolean (default true) display or not warnings on the map
arg.warning_message any (default [])
Returns
SVGSVGElement: the function returns a svg container + some information about this container: projection , margin , width , height and bbox
Related
https://observablehq.com/@neocartocnrs/geoviz
Example
let svg = geoviz.create({width: 500, background: "lightblue"})

The path function generates SVG paths from a geoJSON
